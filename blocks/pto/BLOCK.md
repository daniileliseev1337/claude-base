## Зона блока

Каскады задач ПТО: объёмы → ВОР → спецификации → исполнительная документация → том ИД.
Поэлементная проработка каскадов: где скрипт, где агент, где субагент — на боевых
задачах хаба, затем шлифовка через feedback.
