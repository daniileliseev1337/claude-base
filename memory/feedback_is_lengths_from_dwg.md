Правила пользователя по длинам/размерам на исполнительных схемах трубопроводов (заданы 2026-06-23 на пилотном томе ВиВ, после двух моих ошибок подряд). Wiki-ссылки `[[feedback_is_lengths_from_dwg]]` в трекерах томов ведут сюда — файл восстановлен 2026-07-06 (оригинал был утерян, правило жило только в заметках тома).

**Why:** дважды брал длины не оттуда. Горизонтальные длины на аксонометрии схематичны и врут; внешние замеры монтажника — не источник. DWG-подложка 1:1 и есть факт по размерам.

**How to apply:**
1. Длины участков мерить ПО DWG-плану (подложка 1:1 = факт), НЕ из внешних замеров.
2. Размеры/длины проставлять ТОЛЬКО на ПЛАНЕ, НЕ на аксонометрии.
3. По аксонометрии считать только ВЫСОТНЫЕ длины вертикальных участков (верхняя отметка − нижняя) и фасонку (опуск «Г» = 2 отвода Ду + вертикальный опуск N м).
4. Σ метража по Ду из ВОР — самопроверка, не эталон: расхождение Σфакт↔ВОР — норма, НЕ подгонять, проговаривать с пользователем.
5. Соседний том — НЕ эталон методики (не копировать, опираться на это правило).

Полный свод практики ПТО: `blocks/pto/ЗНАНИЯ ПТО — практика подсчёта и источники факта.md` (§3–4). Связано: [[feedback_user_rules_docs_cascade]].
