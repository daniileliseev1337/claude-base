## Updater 2.0 — для сотрудников при настройке нового ПК

`~/.claude/scripts/Update-ClaudeBase.bat` — **двойной клик в проводнике**.
Скрипт автоматически:

- определяет роль (developer/consumer);
- делает `git pull`;
- вливает shared settings в личный settings.json;
- запускает 22-23 проверки;
- (для consumer) интерактивно создаёт `.feedback-config.json` если его нет;
- (для consumer) делает smoke-test push в `claude-base-feedback`;
- показывает итоговый PASS/FAIL по шагам с конкретной диагностикой.

Это **первое действие** на новом ПК после клона `~/.claude/` (или при
любом сомнении что что-то рассинхронизировалось). Заменяет ~8 ручных
команд одной кнопкой.

См. CHANGELOG.md 2026-05-22 для детального описания.
