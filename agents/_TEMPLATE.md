## After completion

<Что делать после выполнения работы. У нас стандарт:
artifact-→-orchestrator-→-reviewer-→-user, не agent-→-user напрямую.>

Когда артефакт готов:

1. **НЕ выдавать пользователю напрямую.**
2. Вернуть orchestrator'у (основному Claude) с пометкой «готово для
   аудита» + структурированный output (см. «Output format» выше).
3. Основной Claude спавнит соответствующего ревьюера:
   - `auditor` (общий) для большинства артефактов.
   - `word-checker` / `excel-validator` / `pdf-reviewer` (узкие) по
     формату артефакта.
4. Артефакт доходит до пользователя **только** после PASSED-аудита
   или с явной эскалацией BLOCKER на пользователя.
