# Агенты

Спецагенты для конкретных задач. Каждый — отдельный файл с описанием, тулзами и поведением. Вызываются через Task tool с `subagent_type = <имя файла>`.

## Каталог

- [[designer]] — проектирование инженерных систем зданий (ОВ/ВК/ЭО/СС). Концепция, расчёты, подбор оборудования, чертежи.
- [[auditor]] — независимый ревьюер артефактов. Получает только ТЗ + финальный артефакт, без хода рассуждений. PASSED/NOT PASSED.
- [[excel-validator]] — read-only валидация Excel. Канал 1 (Excel MCP), Канал 2 (openpyxl/pandas fallback). См. [[2026-05-18_lesson-15-proxy-helpers-persistence]] как пример «закрытия задачи».
- [[word-checker]] — read-only ревью DOCX (структура, плейсхолдеры, форматирование). Используется после генерации по шаблону.
- [[pdf-reviewer]] — read-only ревью PDF (метаданные, аннотации, формы, шрифты). После любой существенной правки PDF.
- [[pyrevit-engineer]] — код-агент PyRevit-расширений К-7 (IronPython + Revit .NET API): багфиксы script.py, новые кнопки, иконки, документация. Спроектирован экономно по токенам. 16-й агент (с 2026-06-03).

### Доменные writer-агенты (с 2026-05-25)

- [[pto-engineer]] — инженер ПТО: ВОР, спецификации, разделы ПД/РД.
- [[сметчик]] — локальные сметы КС-2/КС-3, расценки ГЭСН/ФЕР.
- [[снабженец]] — УПД, сравнение цен поставщиков, сверка со спецификацией.
- [[kp-writer]] — коммерческие предложения заказчику.
- [[letter-writer]] — деловая переписка (исх. письма по ГОСТ Р 7.0.97-2016).
- [[id-engineer]] — исполнительная документация (АОСР/АОМР/журналы, РД 11-02/11-05).
- [[expertiza-responder]] — ответы на замечания экспертизы (ПП 87, ПП 145).

### Доменные ревьюеры (read-only, с 2026-05-25)

- [[norm-lookup]] — поиск точных цитат по ГОСТ/СНиП/СП/МДС/ПУЭ (нормы только через него).
- [[audit-rd-section]] — нормоконтроль раздела РД/ПД на ГОСТ СПДС, 87-ПП, ПУЭ.
- [[rd-coordinator]] — кросс-секционная согласованность РД (ОВ↔ЭО↔ВК↔КР).

## Когда вызывать

Перед выдачей артефакта пользователю — обязательно соответствующий read-only ревьюер (см. [[CLAUDE]] секцию «Агенты-проверяльщики»).
