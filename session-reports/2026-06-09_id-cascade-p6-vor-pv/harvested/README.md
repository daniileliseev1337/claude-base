# Harvested — каскад п.6 (ИД ВиВ)

## id_cascade_tool.py
Обобщённый, обезличенный инструмент вставки позиций в листы ИД-книги (ПВ/ВОР) с сохранением
merge и перенумерацией по прогонам. Параметризуется блоком `CONFIG` (путь, ITEMS по помещениям,
схемы листов). Выжимка из рабочего скрипта прогона 2026-06-09 (3 листа, 8 сифонов / 16 подводок,
QA excel-validator + auditor PASSED).

**Кандидат → `skills/id-cascade/tools/`** (3-слойный скилл: Description=когда каскадить +
Instructions=шаги chain `id-volume-cascade` + Tools=этот скрипт). Предложено пользователю.

## Проверенные инварианты дисциплины (почему именно так)
- merge снять → вставка снизу-вверх → восстановить со сдвигом `shift(r)=Σ вставок с границей ≤ r`;
- перенумерация по прогонам (reset на строке-шапке) — чистит и старые дубли/дыры;
- стиль и №Акта/Дата/Ед.изм — копировать у соседа (последняя строка блока), не выдумывать;
- идемпотентность (abort, если позиция уже есть);
- после: открыть в Excel и пересохранить (openpyxl не кэширует значения формул).
