# Отчёт сессии 2026-07-05 — Revit АР 2D→3D (ресторан, боевой проект)

## Задача
Возвести АР (стены+двери) из 2D-подложки в 3D поверх готовой ОВ-модели; отработать метод
«2D-линия → 3D-стена+дверь» на одном помещении (тамбур), затем масштабировать.

## Сделано
- Карта понимания (understanding-map) виджетом + HTML — сверка до старта.
- Метод **отработан на тамбуре**: 4 стены `Wall.Create` по осевым из подложки + проём двери
  `NewOpening` по ведомости. Первый `commit=Committed`.
- Создан README инфраструктуры Revit в рабочей папке; обновлена память проекта и метода.

## Источники
- Комплект ПСД (Я.Диск, public API `cloud-api.yandex.net/v1/disk/public/resources`).
- PDF АР «Монтажный план перегородок» — экспликация, ведомость проёмов, толщины (`pdf_render_pages` + clip).
- Живая модель через MCP Revit-Connector (`execute_revit_code`).

## Повороты / где ошибся
1. Память говорила «номера помещений в 45 TextNotes» — оказались **легендой вентиляции**, не номерами.
2. Отдельный свежий **АР DWG — грязный Archicad-экспорт** (2.4 км разброс, слои в '0', СК не совпала
   origin-to-origin). Отклонён и удалён после импорта. Урок: сначала проверять bbox/слои перед ставкой на источник.
3. Разгадка привязки: **сетка осей нашлась в самой вент-подложке** в координатах модели → PDF↔модель связаны.
4. Локализация помещения по первому вопросу ушла не туда (принял служебный вход за главный) — исправил по факту с плана.
5. Мой вывод «библиотеки семейств нет» **неверен** — она на отдельном диске (корпоративная, 2648 rfa);
   зафиксировал в README по просьбе владельца.
6. Дважды попал в unicode-ловушку: кириллицу в литералах `execute` писать ТОЛЬКО как `\uXXXX`-escape.

## Уроки (в память)
- Источник геометрии АР = существующая подложка (несёт оси в СК модели), а не отдельный грязный DWG.
- Метод: извлечь линии `get_Geometry` → сгруппировать в стены (вертикали/горизонтали, merge) → `Wall.Create`.
- Скриншот плана с кириллич. именем вида → создать ASCII-дубликат `ViewPlan.Create`+crop.
- LoadFamily — только из корпоративной библиотеки на отдельном диске (стандартная Autodesk не установлена).
