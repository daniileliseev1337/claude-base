- **Описание:** self-hosted «агентский хаб» поверх готовых agent-runtime'ов (OpenCode, DeepAgents, Hermes, Claude Managed Agents, Cursor Agents API): создание агентов в UI, единый API, сессии, CRON-расписания, память, skills/tools. Rust+TypeScript, docker compose.

Пользователь прислал после развёртывания LiteLLM-гейтвея — кандидат на «агентский хаб» для локальных моделей.
