## Оценка
- Подходит? Под условием — как ДОПОЛНЕНИЕ, не замена; только через пилот на dev-ПК
- Сильные: закрывает дыру «сессия без отчёта ушла в никуда» (автозахват); поиск по ВСЕЙ истории сессий; веб-вьюер = наблюдаемость, которую просит пользователь; Apache-2.0; Bun уже стоит (бот)
- Слабые/риски: (1) ставит СВОИ хуки SessionStart/PostToolUse/SessionEnd — прямой риск конфликта с нашими auto-pull/auto-push/телеметрией (телеметрию уже теряли при мерже конфигов 10.06); (2) компрессия observations жрёт токены на каждом Stop/SessionEnd — на фоне инцидента 100M/день считать обязательно; (3) память per-machine в SQLite — НЕ шарится между 9 ПК (наша — через git); (4) обезличивание: observations пишут сырое содержимое инструментов; (5) крипто-токен CMEM в README — флажок культуры проекта
- Гипотеза для пилота: проверить, можно ли компрессию повесить на локальный Ollama (OpenClaw-инсталлер упоминает выбор AI provider) → 0 токенов
