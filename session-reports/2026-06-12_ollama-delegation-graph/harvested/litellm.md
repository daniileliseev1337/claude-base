- **Описание:** индустриальный стандарт LLM-гейтвея: единый OpenAI-совместимый endpoint поверх 100+ провайдеров (включая Ollama), роутинг, fallback, бюджеты, логирование, admin-дашборд

## Зачем смотрели
«Агентский хаб» на перспективу: роутинг задач между несколькими локальными моделями (qwen3.5 / glm-ocr / qwen3-vl / embedding) + учёт, единая точка для скриптов базы.

## Оценка
- Подходит? Да, когда моделей станет 3+ и появятся постоянные потребители (скрипты, hooks, бот)
- Сильные: зрелость, виден каждый запрос (логи/токены/латентность), Python — родная среда для нашей базы
- Слабые: для одного Ollama-сервера сейчас — оверкилл; полный дашборд хочет PostgreSQL
- Решение: держим в уме как «вторую ступень» после обкатки 2-3 моделей
