# luanshixia/AutoCADCodePack
- URL: https://github.com/luanshixia/AutoCADCodePack
- Stars: 518
- Last commit: 2024-05-26
- License: MIT
- Описание: C#-обёртка над AutoCAD .NET API. Пере-инкапсулирует устаревшие классы в современные статические модули с поддержкой LINQ, сокращая код в 2 раза.

## Зачем смотрели
Сценарий 7 (готовые AutoCAD .NET-плагины как референс).

## Оценка
- **Подходит? Под условием (если разрабатываем .NET-плагин).**
- **Сильные стороны:**
  - MIT, 518 звёзд.
  - LINQ-стиль работы с DWG-сущностями — намного приятнее голого AutoCAD .NET API.
  - Качественный референс для написания собственных плагинов.
- **Слабые стороны / риски:**
  - Последний коммит май 2024 (на грани 12-месячного фильтра, формально проходит).
  - Заявлено для AutoCAD R23/2019 — для AutoCAD 2024+ возможны проблемы.
  - Только для .NET-разработки внутри AutoCAD — не headless.
  - Альтернатива **Linq2Acad** (76★, MIT) — концептуально похож, аналогичная активность.
- **Решение: держим в уме** как референс если возьмёмся писать AutoCAD-плагин для повторяющейся задачи. **Не используем по умолчанию** — Python+ezdxf быстрее и проще для большинства наших задач.
