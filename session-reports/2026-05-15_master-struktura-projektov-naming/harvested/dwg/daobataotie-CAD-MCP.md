- Описание: MCP server для управления AutoCAD/GstarCAD/ZWCAD через COM-автоматизацию из естественного языка (рисует линии, круги, дуги, прямоугольники, полилинии, текст, штриховки, размеры, управляет слоями).

## Зачем смотрели
Сценарий 8 (MCP-сервер для CAD), как альтернатива puran-water.

## Оценка
- **Подходит? Под условием (только если есть лицензионный AutoCAD/GstarCAD).**
- **Сильные стороны:**
  - MIT.
  - Самый звёздный CAD-MCP по статистике.
  - Поддержка **трёх CAD-платформ** (AutoCAD, GstarCAD, ZWCAD) — если в офисе ZWCAD как замена AutoCAD, это редкая фича.
  - Live-режим: рисует прямо в открытом CAD, видно сразу.
  - Слои, штриховки, размеры — полный набор для строительных чертежей.
- **Слабые стороны / риски:**
  - **Требует установленный CAD** (AutoCAD/GstarCAD/ZWCAD). На headless-машине не работает.
  - **Только Windows**.
  - Не умеет читать существующие DWG (только пишет).
  - Полгода без обновлений (последний коммит июль 2025) — менее активен чем puran-water.
- **Решение: держим в уме** как альтернативу. Если puran-water окажется неудобным **или** возникнет реальный сценарий «работаем в живом AutoCAD/ZWCAD» — берём. Для headless-генерации xlsx→DWG не подходит.
