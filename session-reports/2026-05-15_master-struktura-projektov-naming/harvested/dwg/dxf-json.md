# dotoritos-kim/dxf-json
- URL: https://github.com/dotoritos-kim/dxf-json
- Stars: 122
- Last commit: 2026-05-08
- License: GPL-3.0
- Описание: TypeScript-парсер DXF в JSON-структуру. Поддерживает sync/async parsing, URL fetch, работает в Node.js и в браузере.

## Зачем смотрели
Сценарий 2 (read DXF) для JS/TS-инструментов и веб-просмотрщиков.

## Оценка
- **Подходит? Под условием (GPL-3.0).**
- **Сильные стороны:**
  - Современный TypeScript, ESM + CJS.
  - Активный (коммит за неделю до сессии).
  - Богатая типизация — удобно работать с DXF-сущностями.
  - Можно фильтровать по типам сущностей (LWPOLYLINE и т.п.).
- **Слабые стороны / риски:**
  - **GPL-3.0** — вирусная лицензия, флаг для согласования.
  - Только DXF, не DWG.
  - Только чтение.
- **Решение: держим в уме** для веб-инструментов где DXF удобнее в JSON-форме. В нашем основном Python-стеке не нужен — ezdxf полностью покрывает.
