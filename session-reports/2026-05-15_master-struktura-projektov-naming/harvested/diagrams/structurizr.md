# Structurizr (DSL + Java)

- **URL Java:** https://github.com/structurizr/java — 1.1k stars, Apache 2.0, v5.0.3 от 21 Nov 2025 (**archived 28 Mar 2026 — переехало в `structurizr/structurizr`**).

## Применимость к нашей задаче

- **Плюсы:** C4-model для архитектуры ПО — хорошо для системных архитектур.
- **Минусы:**
  - **C4 заточен под software architecture**, не под физическую СКС-топологию шкафов и кабелей.
  - Нет иконок коммутаторов Cisco.
  - Для 200 физических узлов с этажами — не его кейс.
- **Вывод:** **не подходит** для нашей задачи. Подходит для другого: если будем делать архитектурную диаграмму ИТ-сервисов заказчика (приложения, БД, шины) — тогда да.
