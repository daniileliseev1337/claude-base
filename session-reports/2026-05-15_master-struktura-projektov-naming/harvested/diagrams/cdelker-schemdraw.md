# cdelker/schemdraw

- **URL:** https://github.com/cdelker/schemdraw
- **Stars:** ~238 (мало, но домен узкий)
- **License:** MIT
- **Lang:** Python
- **Описание:** Электрические принципиальные схемы (резисторы, конденсаторы, ОУ, цифровая логика). Поддерживает блок-диаграммы и flowchart.

## Применимость к нашей задаче

- **Плюсы:** аккуратная векторная отрисовка, MIT, есть flowcharts (прямоугольники со стрелками).
- **Минусы:**
  - **Не** про сетевую топологию — нет иконок коммутаторов/серверов.
  - Координаты задаются вручную (декларативно «правее на 3 единицы») — для 200 узлов это пытка.
  - Нет автолейаута.
- **Вывод:** **не подходит** для нашей задачи. Хорош для принципиальных электросхем (могло бы пригодиться при работе с ЭО — однолинейки), но не для топологии ЛВС.
