# Harvest: инструменты антиплагиата / детекции заимствований (2026-06-03)

## Зачем
Дипломная работа (.docx, ~70 стр., рус.). Free Plagiat.AI выдаёт нестабильные
цифры (8% / 37% / 84% на одном файле) и НЕ подсвечивает фрагменты. Нужно: найти
заимствованные куски + источник, чтобы переписывать точечно. Желательно локально,
free/OSS, Windows, русский. Бонус — детекция ИИ.

## Вывод
- Локально воспроизвести официальный русский антиплагиат нельзя — индекс закрыт.
- Для НАШЕЙ задачи (найти и переписать заимствованное) у нас уже ЕСТЬ список
  источников из отчётов Plagiat.AI. Надёжнее всего: скачать тексты этих URL и
  локально сопоставить с телом (shingle/семантика) → получить конкретные абзацы.
  Это паттерн SemOrigin, но реализуемый своим маленьким скриптом без установки
  чужого кода (детерминированно, без риска).

  AI-likeness; стоит попробовать в изолированном venv как эксперимент.
