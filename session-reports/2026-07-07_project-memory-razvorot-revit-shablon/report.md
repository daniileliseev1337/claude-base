## Где сломался / уроки
- **Rename `_CLAUDE` → Permission denied при живом Revit.** Handle держался на САМОЙ
  директории `famfab\battle` (файлы внутри свободны) — вероятно cwd процесса Revit после
  диалога ImportSizeTable боевого теста SHUFT. Windows блокирует rename любого предка.
- Локализация держателя: пробный rename каждой подпапки туда-обратно (бинарный поиск
  по дереву). Скан файлов эксклюзивным открытием давал ложные «локи» на read-only
  git-pack'ах клона (UnauthorizedAccessException ≠ lock) и ловушку `$_` в catch
  (ErrorRecord, не файл — сохранять `$file = $_` до try).
- Обход: перемещение детей held-директории разрешено → переезд всего мимо held-ветки,
  хвост battle доехал в конце сессии, когда Revit отпустил handle. Живой Revit не трогал
  (этикет стенда).
