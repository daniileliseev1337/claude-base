## Метод «живая геометрия → карта» (ГЛАВНОЕ, переиспользовать)
1. **Запрос ЖИВОЙ геометрии** из линков через `execute_revit_code` (IronPython в Revit): двери/стены/окна/колонны/сантех/мебель/наши приборы. `RevitLinkInstance.GetLinkDocument()` + `GetTotalTransform().OfPoint()` для координат в host. Host-док СС пуст — всё в линках АР/ВК.
2. **Фильтр ПО ЭТАЖУ обязателен** — иначе все этажи (13 шт) ложатся в одну XY-проекцию (245 «стен» вместо 26). Для стен — `min(Z кривой)` (baseZ ~23700 для 6 эт); для точечных — по `LevelId.Elevation` (надёжнее Z точки) или Z в [22800..24600].
3. **Сантехника = пачка компонентов** (унитаз+инсталляция+2 кнопки+сифон). Фильтр «крупные» по bbox>250мм убирает шум (432→190).
4. **Двери:** ширина в параметре «Ширина» (instance) ИЛИ парс из имени типа `проемШИРИНАхВЫСОТА` (regex `\d{3,4}` — латиница, без кириллицы). BuiltInParameter.DOOR_WIDTH на Symbol даёт 0.
5. **Запись:** IronPython пишет JSON-файл → Python (matplotlib/чистый) генерит HTML с inline-данными → Canvas+JS виджет. Скрипты: `C:\Temp\blsh_sps\build_widget.py`, `plot_live_046.py`.
