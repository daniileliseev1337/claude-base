## Финальный pipeline v12

```python
# === ИСТОЧНИК: подготовить чистый чертёж ===

# 1.1 pikepdf — удалить блок штампа из content stream
# Найти "q 1 0 0 1 X Y cm 0 0 595.32 842.04 re W n" (начало блока штампа)
# Найти ПРЕДЫДУЩИЙ clearing pattern (q 0 0 W H re W n 1 1 1 rg X Y W H re f* Q)
# Удалить body[:clearing_start]
# Удалить /Fm0 из page Resources.XObject

# 1.2 pikepdf — удалить MC с vertical text
# Pattern: /OC/MC<N> BDC ... EMC где Tm = 0 .12 -.12 0
# Это vertical text колонок «Согласовано/Взам.инв.№/Подп.и дата/Инв.№подл.»

# 1.3 PyMuPDF apply_redactions(graphics=2) для рамок и грязи
# Узкие зоны по краям MediaBox (по 10pt) — внешняя рамка
# Узкие зоны на ~14-22pt — внутренняя ГОСТ-рамка
# Зона x=0..200, y между внутр. рамкой и колонкой — «грязь»
# Расширить MediaBox на +100pt для захвата за границей

# === ШАБЛОН: патч плейсхолдеров ===

# 2.1 Для повёрнутых шаблонов (А0, А0x2, А4 книжный) — flatten:
# pikepdf wrap content stream в q [0 1 -1 0 H_old 0] cm ... Q
# Обновить MediaBox на (H_old, W_old)
# Удалить /Rotate

# 2.2 pikepdf — удалить hex-string <НАИМЕНОВАНИЕ> и <Лист>
# Парсить CMap шрифта /F1, найти hex для placeholder, удалить

# 2.3 PyMuPDF insert_text с локальным TTF GOST Common Italic:
# C:\Windows\Fonts\GOST Common Italic.ttf
# point = (cx - text_width/2, y1_placeholder_bbox)
# fontsize = размер плейсхолдера

# 2.4 КРИТИЧНО: doc.save(NEW_PATH, garbage=4) — НЕ incremental!
# Иначе создаётся /Contents Array из N streams = N-кратное наложение

# 2.5 pikepdf merge /Contents Array → 1 single Stream

# === СБОРКА: шаблон-база + чертёж ===

# 3. show_pdf_page чистый источник на patched template
# clip_top = (20, 20, W-20, y_stamp_top)
# clip_bot_left = (20, y_stamp_top, x_stamp_left, H-20)
# Margins 20/35 хватает если source уже чистый (рамки удалены)
```

## Открытые вопросы

1. **Маппинг страниц** — пользователь должен сверить Excel и подтвердить (или править) названия для стр 32-35, 40-42, 70.
2. **«Грязь» на других листах** — на p42 я нашёл 396 мелких vector paths в зоне x<200, y=2130-2310. Удалил через redact. **На других листах** грязь может быть в **других** зонах. Возможно потребуется per-page настройка redact-zones.
3. **A4 книжный с rotation=270** — pipeline для портретных шаблонов с flatten проверен только для А0. Для A4 нужна верификация.
4. **Маленькие штампы (продолжения)** — у меня готовый pipeline только для большого штампа. Для маленьких (форма 6) нужен отдельный pipeline — там нет `<НАИМЕНОВАНИЕ>` плейсхолдера, только `<Лист>` (или его аналог).
