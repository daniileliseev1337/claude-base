## Что внутри

- Парсер CIAN, обход Cloudflare через cloudscraper
- Установка через pip: `pip install cianparser`
- Фильтры: тип сделки (аренда/продажа), количество комнат, регион, и т.д.
- Сохранение в CSV
- НЕ Yandex Realty.
