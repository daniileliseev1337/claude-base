- Задача новой сессии: **спроектировать блок ПТО с ГРАНУЛЯРНЫМИ агентами** (много, на каждую подзадачу), чтобы новый процесс давал чистый результат и не повторял изъяны.
- Метод (утв.): идти **по карте осей Р→Б→1-6**, на каждой переосмыслить ИСПОЛНЕНИЕ + выделить узкие агенты.

## Решение пользователя (дословно по смыслу)
1. Тома Молот/Треугольник больше не правим (Треугольник по качеству ещё хуже Молота — «вообще ужас»; Молот «по более-менее протоптанной дороге»).
2. Создать блок ПТО с агентами, «пусть их будет очень много на каждую подзадачу, но результат чище».
3. Сначала **проговорить каждый шаг** процесса, чтобы отфиксировать ВСЕ изъяны и не повторить их. Потом делать новое правильно.
4. Метод захода (ответ на вопрос): «по карте осей Р→Б→1-6, переосмыслив исполнение».

## ВСЕ пойманные изъяны (главная ценность — чтобы НЕ повторить в новом процессе)
**Ось 2 (документы качества) — самая больная:**
- Реестр-опись: 47 позиций «Технический паспорт / Сертификат качества / Паспорт» **БЕЗ предмета** (на что документ — теряется; предмет писали в колонку «№ документа», не свели). ТН не поймёт.
- Жирность вразброс: жирные поз.1-11 (ок, админблок) + случайные 43-45, 50-51, 100, 109, 113 (наследование формата при вставке строк Excel/openpyxl).
- Нет структуры разделов: плоский список 1-123 без заголовков-группировок (админ/оборуд ВиВ-1/трубопровод ВиВ-3/теплоиз ВиВ-4/заделка ВиВ-5).
**Ось 5 (пересборка) — процессные:**
- Самовольное необратимое удаление 7 стр из юр. тома по находкам автоприёмки БЕЗ согласования (откачено). Правило: находки=кандидаты, изъятие=решение пользователя.
- Эталонная карта приёмки: переданы № позиций описи, агенты приняли за № страниц → 11 ложных CRITICAL.
**Ось 6 (приёмка):** 403-burst на opus fan-out (лечится resume-добор); текстовая сегментация описи невозможна (152 скана).
**Ось Р:** фантомы пересказа замечаний (искажение при переносе — «расход воды» вместо «пены»).
**Ось 3:** scramble листов И.С (текст-слой А3 врёт → ложный PASS); сверять визуально.
**Ось 1:** грабли openpyxl insert/merge (ломает формулы → Excel COM); рассинхрон ПВ↔ВСО.
**Технические (все оси):** fitz.save→tmp+replace; stdout.reconfigure(utf-8); кириллица→forward-slash.
