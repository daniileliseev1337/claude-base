## Ключевые открытия (главная ценность сессии)

1. **ПВ — корень всех фактических объёмов ИД** (оборудование + материалы + изделия, разбитые по
   актам). ВСО (только оборудование) и ВОР — проекции; пользователь исторически нарезал их ИЗ ПВ.
   Прежние формулировки «ВСО главнее» / «источник истины = СВОД+РД» — огрублены и вводят в заблуждение.
2. **Единого проектного файла-эталона объёмов не существует.** Сметы/заключения госэкспертизы в ФС
   нет (проверено глобальным поиском по именам). ИД несёт факт, проектная спека — лишь отправная точка.
3. **«СО для вадима» (Общая спецификация Водоснабжение 27.11.xlsx и пр.) = СТАРЫЕ неактуальные ПВ
   прошлых версий ИД**, а не проектная СО. За проектный источник не принимать.
4. **СВОД оконечки** = 100 % факт, но только несколько позиций и количества на ВСЕ строения → из
   него только наименования/марки, не количества на корпус.
5. **Все вопросы по количествам — проговаривать с пользователем** (по одному, грилинг), не выдумывать,
   не подгонять под сходимость.
6. В заметках тома (`Claude\Р-1-ВиВ…\01_СТАТУС / 02_СОСТАВ / 03_СПРАВКА`) объяснения пользователя
   зафиксированы ПРАВИЛЬНО (подход осевой — неверный, но факты/правила там верные). Использовать
   как справку фактов, не как процесс.
