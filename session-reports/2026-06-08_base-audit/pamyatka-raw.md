# Сырьё для пользовательской памятки (собрано аудитом 2026-06-08)

> 📌 **ИТОГОВЫЙ ФОРМАТ ПАМЯТКИ ДЛЯ ПОЛЬЗОВАТЕЛЕЙ — PDF** (решение пользователя 2026-06-08).
> Собирается ПОСЛЕ 2 больших задач (ИД-сессия + 2 секретных инструмента). Этот файл — сырьё;
> финальная памятка «как пользоваться базой от А до Я» оформляется кратко, просто, с примерами,
> с разделением «по вызову `/`» vs «авто из контекста», и экспортируется в **PDF**.
>
> Готовый материал «что делает + как активируется + пример» для будущей памятки.
> **activation:** `auto` = сам из контекста · `invoke` = только по `/`вызову · `both` = и так, и так.
> ⚠️ Триггеры агентов будут переписаны (правка №1) — после этого примеры станут реально срабатывать.

## Команды (только по `/`вызову — `invoke`)

| Команда | Что делает | Пример |
|---|---|---|
| **/format** | Оформить документ по ГОСТ — выбрать стиль, получить DOCX (опц. PDF) | `/format full ПЗ-ОВ-блок-А` |
| **/harvest** | Ищет готовые open-source инструменты под задачу (GitHub, skills.sh, MCP) | `/harvest extract tables from PDF` |
| **/sync-base** | Подтянуть обновления базы с GitHub + проверить, всё ли доустановлено | `/sync-base` |
