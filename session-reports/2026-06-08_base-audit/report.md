## Уроки сессии (для memory/skill, если решим закрепить)

- **Workflow `args` с кириллицей не доезжает** — большой объект args пришёл пустым (`agents.map` of undefined). Решение: зашивать списки литералами прямо в скрипт, не через `args`. Кандидат в `reference_workflow_tool.md`.
- **Git case-mismatch на Windows:** файл создан строчным, переименован на диске в заглавный — git хранит старое имя, на Linux-клоне расходится. Лечится двухшаговым `git mv`.
- **Две системы памяти** не путать: `~/.claude/memory/` (claude-base, [[wiki]], пушится) vs `~/.claude/projects/.../memory/` (auto-memory, personal, НЕ пушится).
- **Workflow с прямой записью в файлы** + пост-верификация git diff/YAML — рабочий паттерн для batch-правок (агенты/скиллы), но обязательна независимая проверка результата (frontmatter не доверять «на слово»).
