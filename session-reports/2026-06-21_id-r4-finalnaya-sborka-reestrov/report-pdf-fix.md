## Грабли / уроки
- **pdf_render_pages**: `path` (не file_path), pages — СТРОКА диапазона («3»), oversized → рендер на диск (Read PNG) либо `clip=[x0,y0,x1,y1]` (доли 0..1).
- **Сканы-сертификаты** (текст=0) → vision-рендер; размер трубы ТМЗ в поле «Размер D.. S..».
- **Карта страниц сборки ≠ ожиданиям**: визуальная проверка ВСТАВЛЕННОГО (не источника) выявила старые схемы — Smyle на стр.37 тома vs Bau Ceramic в И.С PDF.
- **mtime файлов на Я.Диске ненадёжен** — решали по СОДЕРЖАНИЮ (Smyle vs Bau Ceramic), не по дате.
- **cp1251-консоль** калечит вывод (стрелки/кириллица) — весь лог в utf-8 файл; финальный print падает (exit=1), но работа сделана.
- **openpyxl стирает кэш формул** → пересохранение через Excel COM (wb.Save) восстанавливает; листаж монолита G(n)=F(n-1)+G(n-1), итог 377.
- **Excel COM Rows().Delete** надёжнее openpyxl для удаления строк (merge/формулы/сдвиг сохраняются); строки ищем по № сертификата, не по предполагаемому номеру.
