## Ключевые уроки (переиспользуемые — полные в reference_bylo_stalo_method, грабли 26-33)
- **Диагональный реестр** (блоки не выровнены) + **свопнутые колонки** одного блока —
  парсить по последнему блоку, оффсеты ядра параметризовать.
- **Узлы спеки «Тип NN (компл. N)»**: кол-ва на ОДИН шкаф — множитель узла; доказательство
  сравнением агрегатов (468 = Σ qty×узел = реестр). Обезглавленные узлы (вёрстка) —
  множитель из реестровых титулов «(N к-т)» + open_question.
- **Зоны 2-го уровня** (маркер || узел) — единственный способ раскладки мульти-строчных
  моделей; **построчный chain-план из реестрового apply_plan** (multiset-матчинг) — для
  СВОР-простыни без секций.
- **PDF-спека**: координатный парсер (калибровка по строке цифр-номеров колонок, приклейка
  по Δy со списком для верхних строк, резка рамки словами по x/y) + жёсткая валидация +
  контрольный xlsx для глаз + ручная выборка против PDF.
- **Excel съедает ведущий ноль** цифровой модели при COM-записи — NumberFormat="@";
  числовой verify маскирует.
- **Деньги не трогать автономно**: развёртка щитов «в сборе» (ДС-цены на корпуса) в СВОР
  не внесена — keep + «к просмотру»; вставки без цены в лестнице — ПУСТО + пометка.
