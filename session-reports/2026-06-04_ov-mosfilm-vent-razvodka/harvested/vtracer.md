# vtracer (visioncortex/vtracer)

- **URL:** https://github.com/visioncortex/vtracer
- **Stars:** 6123
- **Last commit (pushed_at):** 2026-03-23
- **License:** MIT
- **Язык:** Rust (есть standalone CLI + WASM + Python-биндинг `vtracer` на PyPI)
- **Описание:** растр→вектор (SVG), цветной, O(n), создавался для гигапиксельных сканов исторических чертежей/синек.

## Зачем смотрели

Нужен общий векторизатор растровой схемы вентиляции (plot-PDF/скан/фото),
чтобы получить snappable вектор как направляющую для трассировки в DWG —
точнее, чем тянуть по растровой подложке «на глаз».

## Оценка

- Подходит? **Да, как быстрый общий вектор-эскиз всей схемы.**
- Сильные стороны: MIT, очень активен, цветной вход (можно отделить
  красный приток от синей вытяжки по цвету), компактный SVG, есть pip-пакет
  (`pip install vtracer`) — установка тривиальна. SVG → импорт в AutoCAD.
- Слабые стороны: даёт **контур** штриха (outline), а не центральную линию —
  толстый воздуховод станет замкнутым прямоугольником-контуром, не осевой.
  Для осей нужен autotrace -centerline или скелетизация.
- Решение: **держать как основной быстрый векторизатор для цветового
  разделения систем и общей геометрии**; центрлайны добирать отдельно.
