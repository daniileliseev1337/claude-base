## Обезличивание

По решению 2026-05-14 репо `claude-base` **private** (на 2026-05-18
public), обезличивание смягчено. Можно: hostnames, email сотрудников
команды, GitHub-аккаунты команды, бренды оборудования, шифры объектов.
Нельзя: пароли, GitHub PAT, ПДн (паспорта, СНИЛС), банковские
реквизиты.

В этом отчёте **есть** (что вошло):
- Hostnames: ...
- GitHub-аккаунты: ...
- Email: ...
- Шифры проектов / бренды (если применимо): ...

В этом отчёте **нет** (что отфильтровано):
- Пароли / API-ключи / PAT
- ПДн третьих лиц (не из нашей команды)
- Банковские реквизиты
- Содержимое, помеченное в моменте `[СЕКРЕТ — не записан]`
