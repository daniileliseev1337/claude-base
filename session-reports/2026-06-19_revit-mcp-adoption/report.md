## Грабли и уроки (полно — `memory/reference_revit_mcp.md`)
- **Кириллица в IronPython exec** — литералы `u"…"` ломаются; идемпотентный хелпер `u()` через `.decode`.
- **safe_tx** с гашением диалогов (модальный диалог вешает Routes-поток → таймаут).
- **Read-only ADSK-параметры** (формульные/каталог) — только правкой rfa.
- **Спеки**: скрытые колонки сдвигают индексы `GetCellText` (принял Массу за Количество); ручные таблицы в Header.
- **Шрифты — 3 уровня**, EditFamily/LoadFamily батчами (таймаут + рассинхрон ответов → маркер-print).
- **Аннотации — обязательный визуальный контроль рендером** (вслепую по координатам — мимо; одна TextNote может иметь несколько leaders → дубли). Вынесено в feedback.
- **Подключение**: корп-прокси (`trust_env=False`), localhost→IPv6 (`REVIT_HOST=127.0.0.1`), Home-экран (нет активного doc), 2 копии main.py.
- **Дистрибуция**: для нового пользователя авто-часть = клиент (скачать/патч/регистрация); Revit-сторона (extension+Routes) остаётся ручной.
