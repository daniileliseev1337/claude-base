## Где сломался / уроки
1. **LISP-счётчики верхнего уровня (`ssget "X"`) недосчитывают графику** — аннотации сидят в блоках, переходы ►◄ сделаны линиями на слое труб → я дал ложный вывод «baseline → не сделано», хотя работа исполнителя была на месте. **Урок (в память `feedback_zoom_before_screenshot`): зум В САМОМ AutoCAD/PDF на зону → потом скрин полного качества; НЕ кропить обзорную картинку** (деградирует). Пользователь дважды указывал.
2. **MCP `plot_pdf` = ok-БЕЗ-файла** (грабля) → плот только через `vla-PlotToFile`; MCP-таймаут на батче не мешает (плот идёт в фоне AutoCAD, проверять по папке).
3. **Облачные плейсхолдеры Yandex.Disk** (демон синхронизации остановлен) блокируют физическую добычу сканов (приказы) — `cloud file provider exited unexpectedly`. Нужна гидратация (поднять клиент).
4. **AutoCAD backend** поднялся только после загрузки `mcp_dispatch.lsp` пользователем в командную строку (до этого ложное «AutoCAD LT detected», хотя запущен полный acad.exe); `system init` подхватил после загрузки.
