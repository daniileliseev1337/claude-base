## Грабли (важные для продолжения)

1. **PATH-клод старый**: `~/.local/bin/claude.exe` = 2.1.114 → «too old for Remote
   Control». Рабочий бинарь: `%APPDATA%\Claude\claude-code\2.1.197\claude.exe` — запускать
   пульт ЯВНЫМ путём.
2. **403 через ~1 час**: сервер получает «Access denied: Request not allowed» при refresh
   токена сессии → сам гасится (архив, чистка environment). Лечение: перезапуск сервера;
   при повторах — авто-перезапуск-обёртка. Пульт — «на время работы», не круглосуточно.
3. **Write-инструмент** отказывает писать в существующий непрочитанный файл («File has
   not been read yet») → выходные артефакты воркеров: уникальные имена или чистка песочницы.
4. Счётчик журнала считал сноски-поправки как прогоны (№5 стал «6») — исправлено:
   считаются только строки с номером во 2-й колонке; время в журнале — человекочитаемое.
5. Sandbox-защита PowerShell ложно матчит inline `python -c` с кириллическими путями —
   выносить код в .py-файл.
