## Ключевые уроки (Family Editor API + MEP)
- Кириллица в IronPython — через файлы (bridge), НЕ литералы (latin-1).
- formula-параметр НЕЛЬЗЯ labeled-dim (конфликт driving); driving-params + значения по типам.
- radial dim постфактум на готовой окружности → «Зависимости не выполняются»; параметрику радиуса —
  через revolve профиль (линейный dim высоты).
- flex-тест через fm.Set(param) ПОРТИТ значения типа → flex ТОЛЬКО сменой CurrentType.
- Connector.Direction READ-ONLY; ошибка на read-only в транзакции → транзакция ВИСНЕТ (GC не откатывает) →
  работать с копией файла.
- Duct.Create берёт систему из аргумента mst (НЕ наследует коннектор); ExhaustAir ≠ ReturnAir.
- MechanicalSystemType.Name падает → DB.Element.Name.GetValue(e).
- ModelText: position = ЛОКАЛЬНЫЕ u/v плоскости; размер = «Размер текста» (кириллица).
- MEP-семейство ОБЯЗАТЕЛЬНО проверять построением разводки; расположение патрубков сверять со схемой источника.
