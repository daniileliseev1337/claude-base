# carnworkstudios/schema-editor

- **Stars:** не подтверждено (org carnworkstudios, репо создан 2025-11, push 2026-06-17 — активен)

- **Язык:** Vanilla JS + SVG, zero dependencies
- **Описание:** Бесплатное web-приложение для проектирования/просмотра электрических схем,
  PCB, network-диаграмм и **архитектурных floorplan'ов** с MEP-символами.

## Зачем смотрели

Vanilla SVG, zero-deps, и прямо «домен» — floorplan + MEP/электрика (наши СС-приборы на плане):
layers, measurement (метрика/имперская), precision snapping, export SVG/PDF/JSON, манхэттен-роутинг.

## Оценка

- **Подходит как зависимость? НЕТ** — это готовое ПРИЛОЖЕНИЕ (не встраиваемая библиотека),
  метрики/лицензия не подтверждены, молодой проект.
- **Подходит как РЕФЕРЕНС? Условно ДА** — посмотреть, как на vanilla SVG без фреймворка
  сделаны слои + измерения + снаппинг + экспорт для инженерных схем (наш профиль).
- Риски: без подтверждённой лицензии код НЕ копировать (только идеи своими словами).
- **Решение: держим в уме как референс vanilla-SVG-подхода.** Сначала уточнить лицензию.
