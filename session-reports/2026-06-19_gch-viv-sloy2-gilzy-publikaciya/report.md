## Грабли autocad-mcp file_ipc (продублированы в память `autocad_mcp_file_ipc_traps`)
- Чтение **свойств ACAD_TABLE через VLA** (`vla-get-InsertionPoint`/`Rows`) **вешает мост** (timeout на всех запросах) → восстановление `system init` + `(princ "ALIVE")`. `vla-GetText`/`vla-SetText` (текст ячеек) — стабильны.
- **PNGOUT из paperspace** (визуал листа) заработал после `system init` (до этого — чёрный кадр).
- Позицию текста на листе привязывать к **рамке чертежа** (по длинным LINE), НЕ к границе VP#1 — у разных форматов (A1/A3/мелкий) даёт разный отступ, текст уходит за область печати.
- **Пакетный плоттинг/PUBLISH одной командой** timeout'ит → печатать партиями ≤4-5 листов (`vla-PlotToFile`, BACKGROUNDPLOT=0) + склейка **pikepdf** (pypdf не установлен).
- Путь с кириллицей в Python — через **файл-скрипт utf-8** (`Write`→`python file.py`), не инлайн `bash -c` (путь ломается).
