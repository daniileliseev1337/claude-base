## Зачем смотрели

Альтернативный PPTX MCP с большой базой пользователей. Не покрывает уникальный сценарий document-loader (PNG-рендер), поэтому как полная замена не подходит.

## Оценка

- **Подходит? Нет (для текущей задачи).** Если потребуется генерация PPTX из шаблонов с богатым форматированием — пригодится. Но для PNG-рендера или универсального парсинга — нет.
- MIT-лицензия, активный, много звёзд — кандидат если в будущем появится задача «генерация PowerPoint-презентаций по ТЗ».
- **Решение: отбросили для текущей задачи, оставили в заметках как ресурс для задач генерации PPTX.**
