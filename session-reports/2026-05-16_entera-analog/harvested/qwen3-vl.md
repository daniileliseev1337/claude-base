# Qwen3-VL (включая Qwen2.5-VL)

- **URL:** https://github.com/QwenLM/Qwen3-VL
- **Категория:** Vision-LLM с локальным запуском
- **Stars:** ~19.2k
- **Last commit:** свежий (релизы 2026 активные)
- **License:** Apache-2.0
- **Описание:** Семейство мультимодальных LLM от Alibaba. Размеры от Qwen3-VL-2B до 72B. Расширенный OCR для 32 языков, устойчивый к низкому освещению/блюру/наклону.

## Зачем смотрели
Главный кандидат на роль «локальной vision-LLM для fallback парсинга УПД через промпт».

## Оценка
- **Подходит?** Да
- **Сильные стороны:**
  - **Apache-2.0**
  - Известный автор (Alibaba/Qwen team — серия моделей в топ-10 open mlllm на 2026)
  - 2B-вариант запускается на скромном железе (RTX 4060 / даже CPU терпимо)
  - Заявлено улучшение информации-извлечения для «qualification review and financial business» — именно наш сценарий
  - 32 языка OCR (русский почти наверняка есть)
- **Слабые стороны / риски:**
  - Llama.cpp/Ollama явно не упомянуты в основном README — нужны конвертация в GGUF (Qwen2.5-VL уже есть в Ollama, Qwen3-VL — на момент 2026-05 проверять)
  - Без fine-tune может галлюцинировать на специфике УПД-нумерации/реквизитов
  - Зависимость от prompt engineering — нужна работа по подбору промпта
- **Решение:** Кандидат №1 для категории «Vision-LLM». Используем как fallback или main parser (через NuExtract-стиль JSON-схему промпта).
