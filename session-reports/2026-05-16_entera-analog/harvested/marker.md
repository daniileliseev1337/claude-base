# marker

- **URL:** https://github.com/datalab-to/marker
- **Категория:** OCR / Document Understanding (PDF → Markdown)
- **Stars:** ~35.2k
- **Last commit:** 2026-01-31 (v1.10.2)
- **License:** GPL-3.0
- **Описание:** «PDF → markdown/JSON», быстро и точно. Использует Surya под капотом + опциональный LLM-постпроцессинг.

## Зачем смотрели
Готовый высокоуровневый конвертер PDF → markdown с таблицами и формулами. Альтернатива MinerU.

## Оценка
- **Подходит?** Под условием (та же лицензия что у Surya)
- **Сильные стороны:**
  - Очень популярен (35k+), хорошая документация
  - Поддерживает PDF/PPTX/DOCX/XLSX/HTML/EPUB — может стать единым входным каналом для всех типов первички
  - **Hybrid mode** с LLM-улучшением точности таблиц — критично для таблиц УПД
  - Отдельный TableConverter
  - JSON schema-driven extraction (можно сразу под формат УПД делать схему)
- **Слабые стороны / риски:**
  - **GPL-3.0** — тот же риск что у Surya
  - Зависит от Surya внутри (если что-то ломается там — ломается и здесь)
  - В hybrid mode требует LLM (внешний API или локальный) — добавляет звено
- **Решение:** Держим в уме как fallback / альтернативу MinerU. Если MinerU будет давать сбои на русских формах — попробовать marker. Лицензия — главный минус.
