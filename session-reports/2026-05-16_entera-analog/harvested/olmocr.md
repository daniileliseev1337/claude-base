# olmocr

- **URL:** https://github.com/allenai/olmocr
- **Категория:** OCR / Document Understanding (для подготовки LLM-датасетов)
- **Stars:** ~17.3k
- **Last commit:** 2026-03-12 (v0.4.27)
- **License:** Apache-2.0
- **Описание:** Toolkit от AllenAI для конвертации PDF/PNG/JPEG в чистый markdown с reading order, формулами, таблицами и рукописным текстом. Специально для подготовки LLM-обучающих датасетов.

## Зачем смотрели
Очень качественный OCR от AllenAI (известный автор — зелёный свет даже на меньших звёздах). Может быть полезен для batch-обработки исторического архива УПД до перехода на MVP.

## Оценка
- **Подходит?** Под условием (требует GPU)
- **Сильные стороны:**
  - **Apache-2.0**
  - AllenAI — серьёзная репутация
  - Обучен на разнообразных документах включая handwriting (рукописные пометки на УПД)
  - Стоимость ~$200 на миллион страниц (для self-hosted) — очень дёшево для backlog-обработки
  - Активно развивается (последний релиз март 2026)
- **Слабые стороны / риски:**
  - **Требует серьёзный GPU** (RTX 4090 / L40S / A100 / H100 с 12+ GB VRAM) для локального inference
  - Русский явно не упомянут в документации
  - Заточен под «архивный батч», не под realtime-обработку одного УПД
- **Решение:** Держим в уме для batch-обработки исторических архивов (если такая задача всплывёт). Для realtime MVP — overkill, требует GPU.
