# MiniCPM-V

- **URL:** https://github.com/OpenBMB/MiniCPM-V
- **Категория:** Vision-LLM с локальным запуском
- **Stars:** ~25k
- **Last commit:** 2026-05-17 (API service release v4.6/o4.5)
- **License:** Apache-2.0
- **Описание:** «Карманная» MLLM от OpenBMB. V4.6 = 1.3B параметров, превосходит Gemma4-E2B. O4.5 = 9B, vision+speech+full-duplex live streaming.

## Зачем смотрели
Самая маленькая (1.3B!) производительная vision-LLM с поддержкой llama.cpp и Ollama. Реально может работать на бухгалтерском ПК без GPU.

## Оценка
- **Подходит?** Да
- **Сильные стороны:**
  - **Apache-2.0**
  - **Поддерживает Ollama и llama.cpp нативно** (GGUF готов: 2GB для V4.6)
  - 1.3B — реально на CPU за секунды на страницу
  - 30+ языков, English document parsing — SOTA для своего размера
  - Активное развитие (релиз в мае 2026)
- **Слабые стороны / риски:**
  - 1.3B всё-таки маленькая модель — на сложных табличных формах УПД может проигрывать Qwen3-VL-2B
  - Заточена под мобильные сценарии (картинки с телефона), не под сканированные документы — надо тестировать
  - Русский в 30+ языках — да, но без специальной отметки по качеству OCR на кириллице
- **Решение:** Кандидат №2 для категории Vision-LLM. **Главный плюс — нативная Ollama-обвязка** = быстрый прототип за день. Идеален для MVP, можно потом мигрировать на Qwen3-VL-2B при необходимости.
