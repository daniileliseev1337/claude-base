## Уроки
- **Хуки/strictly-shared ключи — только в `settings.shared.json`**; personal затирается
  merge'ем при каждом старте (memory `settings_hooks_shared_merge`).
- Маски заголовков журнала должны требовать дату (`^## \d{4}-`): ручные шапки журналов
  законно содержат `## `-строки в код-блоках примеров.
- Сверку «инструмент ↔ ручная структура» делать прогоном на копии реальных файлов
  (scratchpad), а не по описанию.
