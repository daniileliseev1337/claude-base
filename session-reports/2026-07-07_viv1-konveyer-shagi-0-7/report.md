## Уроки (уже зашиты в базу)
1. **doc-finder:** обрыв TCP с иностранного egress = геоблок, НЕ «документа нет» и НЕ «на другую
   машину»: зеркала → ru_fetch (годится и для коммерческих) → Wayback (отдаёт исходные байты).
   Субагенту-добытчику прописывать лестницу явно (мой промпт этого не сделал — агент встал).
2. **Канон дат** документов качества: пересечение с периодом работ; эталон-кейс записан в
   ЗНАНИЯ §11 + методичку р.26 (сделала параллельная сессия по этим же материалам).
3. **Решение-принцип владельца** («старые письма оставляем, замечание → заменим») действует на
   категорию — не выносить однотипный частный случай на новую развилку (memory-файл).
4. **Плоты из ACAD (SHX-шрифты)**: текст-слой битый (cid-коды) → координатные парсеры дороже
   пользы; читать клип-рендерами по якорям таблиц. Якоря «Ведомость…» ищутся текстом надёжно.
5. Финансово-тяжёлый субагент (383k токенов на добычу) — нормально для веб-охоты, но лестницу
   дожима надо давать в промпте сразу, иначе итерация повторяется в main.
