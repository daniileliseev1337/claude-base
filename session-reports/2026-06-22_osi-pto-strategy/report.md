## Где сложно / уроки (харвест)
- **Лист ≠ страница PDF** в сканированном теле тома (тело = сканы, текст-слой только в описи) → листаж описи деферить к пересборке (там строится карта страниц). Минорные сверки со сканов (аномальные даты) — тоже к пересборке.
- **Гарды-перед-сохранением (all-or-nothing) поймали ошибку индексов строк (+2)** при правке описи — сохранения не было, исправил по верной формуле `строка = поз + N`. Паттерн «guard-substr на каждую ячейку + сохранять только при 0 FAIL» окупился.
- **Холодный uvx-кэш office-MCP:** ревьюер с ТОЛЬКО excel-MCP (excel-validator) читать не сможет; брать `auditor` — у него есть Bash → python-фолбэк (openpyxl/python-docx). Подтвердился анти-холодный-кэш принцип CLAUDE.md.
- **Инсайт пользователя (важный):** ни одна ось/блок ПТО не знает, в каком виде том был ИЗНАЧАЛЬНО — все знают только замечания. Решение — **Ось Б** (независимый аудит оригинала) как грунт под все оси.
- **Принцип зон:** ось, нашедшая чужое расхождение, пишет блокер и НЕ чинит сама (иначе двойные правки и сломанный инвариант). Переход между осями гейтит пользователь при нуле кросс-блокеров.
