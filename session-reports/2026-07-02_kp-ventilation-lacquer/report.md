# Сессия 2026-07-02 — КП на проектирование вентиляции (линия лакировки)

## Что сделал
- domain-grilling по вводным КП (вопросы по одному, с рекомендациями): объект — производство
  (намотка обмоток трансформаторов, линия лакировки, пары растворителей), Москва/ближнее
  Подмосковье; состав — одностадийная РД марки ОВ (местные отсосы-зонты 6–12 шт + приточная
  компенсация) + этап обследования существующих вытяжных систем + доп. выезды.
- Ключевая вводная: у исполнителя пока НЕТ фирмы и прайса → все цены в КП только вилками
  с пометкой [ОРИЕНТИР], выведены из рыночных данных (веб-поиск exa, 5+ источников).
- kp-writer (sonnet) собрал .docx: 7 разделов, таблица стоимости 5×4, нейтральная шапка
  без реквизитов (плейсхолдеры <Заказчик>, <ФИО, контакт>), без цитат норм по памяти.
- Ревью: auditor — PASSED (15/15), word-checker — WARN (2 пустые ячейки в ИТОГО).
  Пустые ячейки закрыты прочерками (python-docx). Расхождения вердиктов не было.

## Итоговые цифры (ориентиры, требуют утверждения пользователем)
- Этап 1 обследование: 35–70 тыс. ₽, 1–2 недели.
- Этап 2 РД марки ОВ: 150–350 тыс. ₽, 4–6 недель.
- Этап 3 доп. выезды: 5–10 тыс. ₽/выезд, в ИТОГО не входит.
- ИТОГО: 185–420 тыс. ₽.

## Где сломался / уроки

  только вручную в Word (известное ограничение ревьюера).

  — грилинг корректно свернул ветку, не настаивал.
