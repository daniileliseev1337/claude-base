## Уроки для будущих сессий

1. **word MCP `add_heading` не поддерживает `color`** — попытка передать `color` ломает создание заголовка. Использовать `add_paragraph` для цветного текста.
2. **Когда `data` не совпадает с `rows`** — pустая строка в конце или обрезка последних. Всегда сверять `rows == len(data)`.
3. **Опечатки кириллицы/латиницы при копировании** — «P» вместо «П», «Act» вместо «Акт», «ъ» вместо «а» и т.п. После генерации текста с кириллицей пройти grep'ом по подозрительным сочетаниям.
4. **`insert_header_near_text` иногда не удаляет дубль** — если функция вставляет, проверять outline через get_document_outline.
5. **word-checker subagent не имеет word MCP** — для реального ревью docx использовать `get_document_text` + `get_document_outline` в main session.
6. **Counts/summaries обязательно сверять с перечнем** — несовпадение «9 принятых» / «8 номеров» = ловушка. word-checker сработал именно на этом.
