# Автономный конвейер «было-стало → СВОР/ВОР» по 4 разделам (09.07.2026)

## Задача
Продолжение задачи версионных таблиц «было-стало» и цепи СВОР/ВОР (объект `<объект>`,
поставка `<договор>`). Автономный режим (пользователь отсутствует, вопросы задавать некому,
неоднозначности помечать «к просмотру», не блокировать конвейер). Два блока:
1. Дельта-ревью слоя наименований (гейт правок, внесённых в принятые файлы прошлой сессией).
2. Конвейер по 4 разделам с готовыми спеками (СПС-МГН, СКПТ, ЭЧ, СТМ), per-раздел:
   адаптер спеки → обёртка ядра → diff → decisions → plan/apply → fix-merges → verify →
   слой имён → цепь → ревью-гейт → дашборд → FACTS.

## Ключевые уроки (переиспользуемые)
- **Глобальная зона** для разделов, где спека — плоский агрегат, а реестр распределён по
  шкафам/корпусам (ЭЧ часы, СТМ телемедицина): одна зона на весь раздел, агрегат по модели,
  раскладка сохраняется где суммы равны.
- **Заглушка-код «-»/пустое** разъезжает агрегат (одна позиция под двумя ключами) —
  нормализовать «-»→None перед diff; в цепи — и в *_blocks.json, И в самих файлах (verify
  перепарсивает файлы).
- **`import bs_ms` ребиндит глобали ядра** (B.BS_FILE/B.WORK) — восстанавливать перед вызовом
  переиспользуемой функции (fix-merges молча правил чужой файл — поймано на СТМ merge-хвосте).
- **Хуки ядра аддитивно**: QTY_KEEP (единица уп.↔м — не менять метраж), ZERO_KEEP (спорную
  «лишнюю» не обнулять), CHAIN_VERIFY_SKIP (осознанные исключения из инварианта) — пустой
  дефолт нейтрален для принятых разделов.
- **verify ВСЕГО** падает, если реестр хранит итог числом, не формулой — ветка тождества значений.
- **Вставка без артикула** → нет шаблона → apply SKIP: пост-патч донор-блок; цены нет в
  ДС/рынке → ПУСТО + пометка, не выдумывать.

## Ревью-гейт
Профильный (мой детерминированный verify + check_highlight = роль excel-validator по форме) +
общий auditor (sonnet, независимый пересчёт денег и содержания). Замечание ревьюеров «не смог
проверить подсветку — нет Bash» трактовалось как граница инструмента (недонастройка спавна),
закрыто своим скриптом, НЕ как дефект артефакта. Все содержательные вердикты — PASS.

## Где споткнулся
