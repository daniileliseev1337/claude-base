# 0xabu/pdfannots

- **URL:** https://github.com/0xabu/pdfannots
- **Stars:** ~700 (на 2026-05)

- **License:** MIT
- **Описание:** CLI tool, извлекает текстовые аннотации (highlight, comment, popup) из PDF в Markdown/JSON.

## Зачем смотрели

В задаче с СЭЧ-спецификацией PDF содержал ~30 аннотаций ПОПРАВИТЬ/ДОБАВИТЬ/УДАЛИТЬ — нужно было программно сопоставить их с позициями таблицы в xlsx.

## Оценка

- **Подходит?** Под условием.
- **Сильные стороны:**
  - Простой CLI: `pdfannots file.pdf -o out.md`.
  - Сохраняет координаты аннотаций (page, x, y) — критично для понимания «к какой строке таблицы относится пометка».
  - MIT-лицензия, можно адаптировать.
- **Слабые стороны / риски:**
  - Работает только с **PDF-аннотациями (Acrobat-комментарии)**. Если пометки нарисованы поверх (как в нашем случае — синий/зелёный/красный текст-аннотации в Distiller'е), могут не быть классифицированы как «аннотации» в PDF-смысле, а быть просто текстом в content stream.
  - В нашей задаче `pdf-mcp pdf_read_all` справился — текст пометок уже был в text layer.
- **Решение:** держим в уме. Если попадётся PDF с **настоящими аннотациями** (комментарии в Acrobat Reader), это **самый быстрый путь**. Для PDF-распечаток с textlayer-разметкой — наш pdf-mcp лучше.
