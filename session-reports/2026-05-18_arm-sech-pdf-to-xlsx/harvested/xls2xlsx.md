# xls2xlsx (pypi)

- **URL:** https://pypi.org/project/xls2xlsx/  (репо: https://github.com/snoopyjc/xls2xlsx)
- **Stars:** ~30
- **Last commit:** 2022 (формально >12 месяцев → флаг)
- **License:** MIT
- **Описание:** Конвертация старого `.xls` (Excel 97-2003) в современный `.xlsx` без Excel/LibreOffice. Pure-Python.

## Зачем смотрели

В этой сессии openpyxl не открывает `.xls` — пришлось конвертировать. Я пошёл через pywin32 + Excel COM. Альтернатива — xls2xlsx, не требует Excel.

## Оценка

- **Подходит?** Под условием.
- **Сильные стороны:**
  - Не требует Excel или LibreOffice — критично если их нет (CI, контейнер, ПК без админ-прав).
  - Простой API: `XLS2XLSX(path).to_xlsx()`.
- **Слабые стороны / риски:**
  - Может потерять стили / условное форматирование / макросы.
  - Последний коммит 2022 — формально не активный (>12 мес).
  - <50 stars — слабый сигнал поддержки.
- **Решение:** держим как fallback. У нас на ПК есть Excel, COM-конвертация работала надёжно — этого достаточно. Если на другом ПК Excel не будет (новичок, легковесный VM), xls2xlsx — первый кандидат.

## Команда установки (если нужно)
```powershell
python -m pip install --user xls2xlsx
```
