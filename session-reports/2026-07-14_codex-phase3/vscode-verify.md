# Верификация Codex-расширения VS Code (Эпик 2, §5 спеки)

Заполняется владельцем или сессией с GUI. Итог — строки матрицы «фича × среда»
(заготовка методички Эпика 6). Спека: проект «Реворк базы»,
`docs/superpowers/specs/2026-07-14-epic2-targets-etalon-design.md`.

Дата проверки: __ · Машина: __ · Версия расширения: __

| Проверка | Как | Результат |
|---|---|---|
| AGENTS.md-правила видны | спросить «какие у тебя правила?» (ожидаем пересказ ядра по-русски) | |
| Субагенты `~/.codex/agents/*.toml` доступны | вызвать любого (напр. auditor) | |
| Junction-скиллы `~/.agents/skills` видны | спросить список скиллов (ожидаем 10) | |
| Плагины documents/spreadsheets/pdf работают | открыть/прочитать тест-файл .docx/.xlsx | |
| Профиль plus/pro выбирается | настройка расширения / запуск с `--profile plus` | |
| Эталонные секции managed-блока действуют | спросить `[agents] max_threads` / проверить поведение | |
| project-AGENTS.md подхватывается | открыть папку проекта «Реворк базы» (в корне AGENTS.md) | |
