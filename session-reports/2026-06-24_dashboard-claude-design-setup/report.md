  Не доверять имени из постановки — проверять через GitHub API/факт.

  структурное к коду (к себе) и принижать новый инструмент Claude Design оговорками. Владелец
  это поймал. Коррекция: дизайн-фазу ведём через Claude Design как основной канал.
