## Где сломался / грабли (→ дописаны в DESIGN р.12)
- **GPU-ASR мёртв:** faster-whisper на CUDA падает — `cublas64_12.dll` не загружается. Лечение: `device=cpu,
  compute_type=int8, model=large-v3`. (GPU-стек CUDA/cuDNN на пилот-машине неполный.)
- **VAD прячет «нет речи»:** `vad_filter=True` даёт 0 сегментов и на речи, и на тишине — неразличимо.
  Контроль: прогон `vad=False` + фильтр галлюцинаций Whisper («Продолжение следует…», «Субтитры сделал…»)
  = маркер тишины. Так отличил «нет речи» от «тихая речь».
- **Кодировка stdout:** PowerShell перенаправляет вывод Python в cp1251 → кириллица в кракозябрах в
  файле фоновой задачи. Лечение: писать результат самим Python в файл `encoding="utf-8"`, потом читать.
- **word `get_document_text` на большом акте** (69 КБ) превысил лимит токенов → сохранён в файл, читать
  порциями. ВСО (тоже крупный) прошёл.
