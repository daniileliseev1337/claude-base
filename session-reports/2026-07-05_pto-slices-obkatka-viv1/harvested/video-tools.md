# Harvest — инструменты обработки видео обхода ТН (2026-07-05)

## Задача
5 видео обхода технадзора (.mp4, ~40 МБ) → русская закадровая речь с таймкодами + кадры с
таймкодами, чтобы связать устное замечание ТН с показанным листом/узлом. Локально/free/приватно
(данные объекта), Windows 11, Python. Эталон сравнения — наш skill `local-video-digest`
(faster-whisper + ffmpeg кадры **по интервалу**, таймкоды сегментные `[mm:ss]`).

## Оценка

**Прошли фильтр (зрелые «кирпичи»):**
- **PySceneDetect** — закрывает главный зазор эталона: кадр берётся в момент смены сцены (ТН
  перевёл камеру на новый лист/узел), а не слепо по таймеру. BSD-3, 5k, живой с 2014. Локально.
  → **брать** как режим отбора кадров.
- **WhisperX** — ASR сильнее faster-whisper: VAD (режет галлюцинации на фоновом шуме стройки),
  word-level таймкоды, опц. диаризация (разделить ТН/прораба, если 2 голоса). Backend = тот же
  faster-whisper. RU: alignment требует wav2vec2-RU модель с HF; диаризация — HF-токен. Тяжелее
  ставится (torch/pyannote). → опция, если нужна точность/шумостойкость.
- **stable-ts** — drop-in улучшение таймкодов whisper (word+segment, refine), MIT, для RU без
  спец-модели. НО репо **архивирован** (заморожен, фиксов не будет). → запасной вариант.

**Отклонены (сырые / приватность):**
- transcreve-ai (1★), video_analysis (1★), vzt-video-intel (0★) — идея «видео→dossier» правильная
  (OCR кадров, timeline, MD/DOCX), но не проверены сообществом, тянут тяжёлый стек (Ollama/Cognee/
  Gemma/Docker), часть стадий облачные (vzt — Qwen/SAM2 на Replicate; berdaflex — Gemma). Для
  приватных данных объекта и разовой задачи — переусложнение (Karpathy: простота). Держим как
  референс-идеи (OCR кадров, формат dossier), код не тащим.

## Рекомендация
Не менять эталон на чужой комбайн. **Усилить `local-video-digest` точечно:**
+ PySceneDetect (кадры по сцене) — главный выигрыш связки реплика↔лист;
+ (опц.) WhisperX для VAD/word-level, если шум/несколько голосов помешают faster-whisper large-v3.
Все компоненты — pip, локальные, BSD/MIT, приватность сохранена.
