3. **Применено ко всем 31 RCU**, провизуалена каждая секция (6.07/6.08 верх+низ, боксы, МГН,

## Где сломался / уроки
- **Эталон снят ВИЗУАЛЬНО НЕТОЧНО:** принял боковую стену за «стену с дверью». Первое правило
  (v5/v6 на стене с дверью) забраковано. Спасло — покомнатное подтверждение пользователем через
  ASCII-preview (тык в схему снимает терминологию) + тест на 1 палате ДО массовой раскатки.
- **Цикл «не ок без конкретики»:** пользователь несколько раз говорил «не так» без номера палаты.
  Разорвано настойчивой просьбой ОДНОГО номера + preview-вариантов. Урок: при «не так» сразу
  требовать конкретную палату/схему-выбор, не гадать (гадание = корень провала прошлых сессий).
- **GetLinkDocument + collector ВЕШАЕТ Revit MCP:** извлечение геометрии блока3 из AR-линка
  таймаутило (>60с), очередь execute копилась — даже `print` не отвечал ~3 мин. Урок: после
  1-го таймаута тяжёлой линк-операции НЕ слать новые (копятся); экспорт кадров безопасен
  (Revit сам рендерит линк, мой код к линку не обращается). Числовой эталон из линка не понадобился —
  хватило визуала + диалога.
- **ExportImage лимит:** при many-image сторона PNG должна быть <2000px; FitToPage расширяет crop
  (захватывает оси/метки) → высота больше расчётной. Держать PixelSize так, чтобы ОБЕ стороны <2000.
