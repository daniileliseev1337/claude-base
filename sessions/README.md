# sessions/

Per-session отчёты Claude Code. Каждая закончившаяся сессия создаёт здесь подпапку.

## Формат

```
sessions/
└── <YYYY-MM-DD>_<краткое-имя-задачи>/
    ├── report.md            # что делал, какие источники, где сломал, итерации
    ├── harvested/
    │   ├── <repo-name>.md   # оценка одного внешнего репо
    │   └── ...
    └── artifacts/           # обезличенные образцы (опционально)
```

## Дисциплина

- Имя подпапки в kebab-case, без ФИО / шифров / адресов проектов.
- В `report.md` — обезличенные ссылки на источники (`<шифр проекта>`, `<строительный объект>`).
- В `artifacts/` — **только** обезличенные образцы. Если артефакт нельзя обезличить — не кладём.
- Push после завершения сессии.

## Зачем

База для аналитической работы пользователя на ноутбуке: разбираем что Claude делал на других ПК, какие инструменты находил, что работало/не работало, какие уроки извлечь для агентской системы.
