- URL: https://github.com/JaidedAI/EasyOCR

- Описание: Ready-to-use OCR с 80+ языками (включая русский) на PyTorch.

## Зачем смотрели
Самая простая в установке альтернатива Tesseract; нативная поддержка русского.

## Оценка
- Подходит для нашей проблемы? **Да** как лёгкий fallback к PaddleOCR.
- Сильные стороны:
  - **Apache 2.0**.
  - `reader = easyocr.Reader(['ru', 'en'], gpu=False)` — одна строка, multilingual.
  - pip install (но требует torch — это ~2 ГБ).
  - Простой API.
  - Активно поддерживается (последний release 1.7.2 — 2024-09, но репо активный).
- Слабые стороны / риски:
  - torch wheel на Windows может требовать админ-прав в зависимости от среды.
  - Чуть менее точен чем PaddleOCR на сложных шрифтах.
  - Релизы редкие — 2024-09 последний tag, хоть и pushed 2025-12.
- Решение: **держим в уме** как простой fallback. PaddleOCR приоритетнее по точности.
