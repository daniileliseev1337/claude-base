# camelot-dev/camelot (активный форк)
- URL: https://github.com/camelot-dev/camelot
- Stars: ~3700 (форк от 3.7k архивного atlanhq/camelot)
- Last commit: 2026 (v1.0.9 released 2025-08-10, активная разработка)
- License: MIT
- Описание: Python-библиотека для извлечения таблиц из PDF; lattice (по линиям) и stream (по выравниванию текста).

## Зачем смотрели
Table extraction из инженерных PDF (штампы чертежей, спецификации, экспликации, кабельные журналы).

## Оценка
- Подходит для нашей проблемы? **Да** для таблиц с явными линиями (типичные штампы, экспликации).
- Сильные стороны:
  - **MIT**.
  - Lattice-метод отлично работает на инженерных таблицах со штамповыми сетками.
  - Конфигурируемые tweakable settings.
  - Активный форк (`camelot-dev`) после архивации оригинала atlanhq в январе 2025.
- Слабые стороны / риски:
  - Только text-based PDFs — на текст-в-кривых не работает (нужен предварительный OCR-pass).
  - Зависит от Ghostscript (доп. установка).
  - Архивный atlanhq всё ещё в большинстве туториалов — путаница для пользователей.
- Решение: **используем** как первый выбор для табличных данных из инженерных PDF с линиями.
