## Применимость к нашей задаче

- **Плюсы:**
  - Тривиально генерить `.mmd` из Python (текстовый шаблон).
  - SVG/PNG/PDF.
  - Mermaid знают почти все разработчики, легко поддерживать.
- **Минусы:**
  - Mermaid слаб для **сетевых** диаграмм: flowchart/sequence/class/state/git — есть. **Network/Cisco — нет специализированного типа.**
  - Через `flowchart` можно рисовать как прямоугольники со стрелками, **но без иконок коммутаторов** и без хороших группировок по этажам/объектам.
  - Для 200 узлов автолейаут (dagre/elk) даст «лапшу».
- **Вывод:** не лучший выбор для нашей задачи. Хорош для CI/CD pipeline-диаграмм и логических флоу, не для СКС.
