# Скиллы

Методологические скиллы — подключаемые слои с триггерами. Каждый скилл это папка с `SKILL.md`, в котором описано когда подключаться и как действовать.

## Каталог

- [[karpathy-guidelines]] — 5 поведенческих принципов (думай прежде чем кодить, простота, хирургические правки, цели/верификация, помощник не подхалим). При конфликте — приоритетнее любого правила в CLAUDE.md.
- [[excel-helper]] — методология работы с Excel (анализ формул, расхождения, валидация, сравнение версий).
- [[word-helper]] — методология Word (find-replace, шаблоны, генерация, экспорт в md/PDF).
- [[pdf-helper]] — методология PDF (чтение, редактирование, аннотации, формы, сравнение).
- [[stroy-formatting]] — применение ГОСТ-стилей к итоговым DOCX/PDF. 4 шаблона в `formatting-templates/`.
- [[image-text-replace]] — замена текста на сканах/JPEG/PNG через PaddleOCR + LaMa inpaint + Pillow render. Pipeline в `pipeline.py`, lazy install зависимостей. Primary mode — LaMa, opt-in fast-fill для batch'а из белых сканов.
- [[chains-pattern]] — методология именованных цепочек (когда и как создавать chains в `~/.claude/chains/`). Триггеры: «составь pipeline», «многошаговая задача», «несколько последовательных шагов». Импорт из К-7 аудита 2026-05-20.

## Как подключаются

Автоматически по триггерам (см. `description` в SKILL.md каждого скилла). Перед действием Claude сначала читает SKILL.md, потом действует.
