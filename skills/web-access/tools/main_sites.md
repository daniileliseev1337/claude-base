# Стратегия поиска: приоритет по ТИПУ источника (не полный список — его не бывает)

Сайтов бесконечно много, исчерпывающий список невозможен. Поэтому это НЕ каталог «все
сайты», а **порядок приоритета ТИПОВ источников** + короткий seed крупных известных
агрегаторов. Отсутствие сайта в seed ≠ «не искать там» — seed открытый, дополняется по ходу.

Claude на фазе «где искать» идёт по ТИПАМ сверху вниз, а не на случайный первый сайт выдачи.

## Порядок типов (сверху вниз)

### 1. Сайт ПРОИЗВОДИТЕЛЯ — первоисточник (обычно есть ВСЁ необходимое)
На офиц. сайте бренда почти всегда: характеристики, паспорт, сертификат/декларация, РЭ,
актуальный артикул. **Как дойти:** по бренду/модели найти официальный сайт производителя
(поиском «<бренд> официальный сайт»), затем внутренним поиском сайта → карточка изделия →
вкладка «Документация/Сертификаты». Домен не угадывать — брать из выдачи/карточки.
Примеры производителей, у кого «часто много всего»: **Eltex** (`eltex-co.ru` — сетевое/
телеком для СС), **Ridan** (`ridan.ru` — преемник Danfoss, ОВ/автоматика). Остальные — по бренду.

### 2. Крупные B2B-агрегаторы — каталог + документы + цены + наличие
Если производитель не дал или нужна цена/наличие. У них карточки с вкладкой документов.
Открытый seed (дополняется):
- `lunda.ru` — инженерка ОВ/ВК
- `santehkomplekt.ru` — сантехника/инженерка (проверять правильный домен — бывает santech.ru)
- `vseinstrumenti.ru` — ВсеИнструменты: инструмент, оборудование, широкий каталог + документы
- `petrovich.ru` — Петрович: стройматериалы, инженерка
- 1001крепёж — крепёж/метизы (домен уточнить из выдачи, не угадывать)
- `santech.ru` — сантехника (паспорта + СС + гигиенические в карточке)
- `teremonline.ru` — инженерная сантехника (ссылку на документ брать ИЗ карточки, не из Google)
- [+ ваши постоянные поставщики — впишите по ходу]

### 3. Госреестры — верификация реквизитов документа
- `pub.fsa.gov.ru` — Росаккредитация (сертификаты/декларации; SPA-API → браузер, [[ru-gov-access]])
- `fgis.gost.ru` — АРШИН (поверка СИ) · `egrul.nalog.ru` — ЕГРЮЛ (реквизиты изготовителя)
- Поиск документа по артикулу с обходом антибота — скилл `doc-finder` (dorks + добыча PDF)

### 4. Прочая выдача / «левые» сайты — как ДОПОЛНЕНИЕ, не вместо главных
Случайные находки тоже бывают полезны (там иногда лежит то, чего нет у производителя) —
но идти туда ПОСЛЕ типов 1-3, а не первым делом. Проверять правильность домена скриншотом.

## Правило вывода (анти-капитуляция — см. SKILL.md)
«Не нашёл / такого нет» можно сказать ТОЛЬКО пройдя типы 1-3 (производитель + пара крупных
агрегаторов + при нужде реестр) и приложив список проверенного. Два случайных сайта из
выдачи, на которые «не смог зайти», = «я сдался», НЕ «этого нет в интернете».

## Нормы (ГОСТ/СП/СНиП/ПУЭ) — НЕ здесь
Нормы — ТОЛЬКО через агент `norm-lookup` (библиотека Я.Диска / cntd.ru), не «главным сайтом».
