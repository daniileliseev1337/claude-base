# <организация> Yandex Disk — структура и пути

## Корневая структура

```
/  (корень Я.Диска)
├── 01_Компания/        ← общие документы <организация>
├── 02_Проекты/         ← все проекты (19 шт)
├── 03_Архив/           ← завершённые проекты + sandbox
└── 04_Отделы/          ← по подразделениям
```

## Шаблон проекта (`02_Проекты/<код>/`)

```
00_Общее/        — паспорт, контакты, реквизиты заказчика
01_КП/           — коммерческие предложения, презентации
02_Договор/      — основной договор, ДС, спецификации
   └── 05_Переписка/    — официальные письма
03_Финансы/      — счета, акты сверки, оплаты, бюджеты
04_ТЗ/           — техническое задание (исходное и согласованное)
05_Чертежи/      — DWG, PDF чертежи (от заказчика и наши)
06_Акты/         — акты выполненных работ
07_Отчёты/       — управленческая отчётность по проекту
08_Архив/        — материалы по закрытым этапам
```

## Полезное

- При создании файла в подпапке, которой нет — `upload_file` автоматически создаст всю цепочку через MKCOL.
- Если код проекта неизвестен — кладём в `03_Архив/inbox_unsorted/` для ручной разборки.
- Для тестов и временных файлов — `03_Архив/skills_smoketest/`.
