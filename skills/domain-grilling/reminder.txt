⚠ domain-grilling доступен для этой задачи (НЕ запускать автоматически).

Запрос похож на строй-задачу (спецификация / ВОР / расчёт / подбор / смета / КП / УПД / письмо / акт-ИД / ответ экспертизе), где домысливание вводных = ошибка в документе.

Сам грилинг НЕ начинай. СНАЧАЛА предложи пользователю выбор через AskUserQuestion с кратким описанием, что делает скилл, и пусть он решит:
- Вариант «Пройти domain-grilling»: ты по одному задаёшь уточняющие вопросы по вводным (объект/шифр, источник данных, числа, формат выхода, нормы, границы), к каждому — своя рекомендация, library/norm-first (что выводится из норм/проекта/эталона — находишь, не спрашиваешь); работу начинаешь только когда все ветки разрешены.
- Вариант «Без грилинга, сразу к задаче»: идёшь обычным маршрутом (роутинг → агент) с теми вводными, что есть.

Действуй строго по выбору пользователя. Решение — за ним, не за тобой.
