# 5 методических профилей ПНР по типам систем

**Профиль для каждого типа РАЗНЫЙ. Не выдумывать удельные числа — согласовывать
через AskUserQuestion + аналог из похожего раздела + norm-lookup.**

## 1. Слаботочка (СКС, ОС, СОТ, СКУД, СПС, СОУЭ, АУГПТ)

- Единица ПНР = **адресное устройство**, функции поштучно:
  `итого = функц./ед. × кол-во устройств из ВОР`.
- Функций на устройство 2–12 (контроллер/прибор/извещатель/розетка…).
- **ЗИП:** в ВОР полное кол-во + пометка «X к монтажу + Y ЗИП» в колонке «Тип
  монтажа»; в ПНР — **без ЗИП** (ЗИП не пусконаладится).
- Объединять одинаковые по составу ПНР разных исполнений (розетки разных форм-факторов,
  табло разных типов) — в одну строку ПНР.
- СКС обычно самый большой по функциям (подсистемы UTP кат.6/6А/оптика). Активное
  оборудование и магистраль — в разделе ММС/ЛВС, не в СКС.
- СПС: ЗКПС = (число зон × 2) отдельной строкой; интеграции СКУД↔СПС↔СОУЭ↔АУГПТ —
  отдельные пункты ПНР центральных приборов.
- ОС: ГЭСНп Сб.2 (АСУ) исключает ОС/ОПС → метод аналогии с СПС + оговорка о
  нормативной лакуне.

## 2. Инженерные ОВ/ВК

- Единица ПНР = **система** (или этаж+блоки), НЕ адресное устройство.
- **Вентиляция:** ~8 функц./система по СП 73.13330.2016 (осмотр → опробование →
  замер расходов → балансировка → герметичность → шум → автоматика → сдача) +
  отдельные группы противопожарных клапанов (СП 7.13130) и балансировочных.
- **Отопление:** единица = этаж+блоки (наладка + балансировка клапанов + коллектор +
  приборы + электро). СП 73.13330.2016 разд.7 (опрессовка п.7.3.1) + СП 60.13330.2020.
- **Кондиционирование:** единица = система (VRF/сплит/прецизионный + дренаж).
- **ВК:** системы + блоки. Водоснаб.: наладка + арматура×N + санприборы×N +
  оборуд.×N. Канализация. СП 73.13330.2016 п.7.2/7.5.
- ⚠ При сомнении в объёме — анализ + расширение (кейс: вентиляция 216→401 после
  учёта клапанов), не защита первой версии.

## 3. Электрика (ЭОМ)

- ⚠ Единица ПНР = **ЛИНИЯ и АППАРАТ**, НЕ «щит» (укрупнение «на щит» бракуется как
  заниженное).
- Щиты — общие операции (~4 ф/щит); аппараты защиты 1×N; кабельные линии 2×N
  (изоляция + «фаза-ноль», **N точно из кабельного журнала**); освещение; заземление.
- ГЭСНп 81-05-01; DALI → Сб.2 (АСУ). ВРУ обычно вне раздела. Источник спеки часто
  `.xls` (xlrd).

## 4. Водяное пожаротушение (ВПВ/АУВПТ)

- ПНР = **испытания + наладка полевых устройств**: ПК DN50, сеть, СПЖ, задвижка с
  контролем, затвор, комплексное опробование 1/система. Оросители/воздухоотводчики —
  укрупнённо.
- ГЭСНп 81-05-02 Сб.2; насосная и ПКА обычно вне раздела.

## 5. Оснащение (ТХ)

- **ВОР на всё, ПНР только на АКТИВНОЕ оборудование** (мебель/портьеры/зеркала/кухня
  пусконаладки не имеют).
- Источник — спецификация .СО по помещениям (authoritative), не плоская сводная.
  ИТ (ПК/ноутбуки/МФУ) за ДИТ — исключать.
