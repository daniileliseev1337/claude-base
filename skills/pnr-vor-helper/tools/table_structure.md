# Структура таблиц ПНР и ВОР (эталон)

## ВОР — Ведомость объёмов работ (9 колонок)

| Кол. | Заголовок | Заметка |
|---|---|---|
| A | Поз. | порядковый номер |
| B | № в ЛСР | ссылка на локальный сметный расчёт (часто пусто на стадии П) |
| C | Наименование работ | в группах-заголовках заполнена ТОЛЬКО эта колонка |
| D | Ед. изм. | шт / м.п. / компл. / система |
| E | Кол-во | |
| F | Ссылка | на лист спецификации / чертёж |
| G | Вес одной ед., кг | заголовок с переносом `\n` |
| H | Высота установки, м | заголовок с переносом `\n` |
| I | Тип монтажа | сюда же пометка ЗИП: «X к монтажу + Y ЗИП» |

## ПНР — Программа пуско-наладочных работ (3 колонки)

| Кол. | Заголовок |
|---|---|
| A | Функции (в группах-заголовках заполнена только эта) |
| B | Кол-во функций на единицу |
| C | Кол-во функций итого |

**Формула:** `C = B × (кол-во устройств этого типа из ВОР)`.

## Группы-заголовки и иерархия

- Группа-заголовок = строка, где заполнена только A (ПНР) или C (ВОР), прочие пусты.
- **1 уровень** (типовой для слаботочки/электрики): тип оборудования
  («Ядро ЛВС», «Точка беспроводного доступа», «Контроллер СКУД»).
- **2 уровня** (сложные системы): помещение → подсистема
  («Операционный зал №NNN» → «Подсистема отображения информации»).

## Стиль xlsx (нейтральный)

- thin border на всех ячейках; header **bold**; group — *italic+bold*.
- Шапка строки 1 с переносами `\n` в колонках веса/высоты.
- НЕ синий Claude-стиль (см. excel-helper). Наследовать стиль книги-эталона.

## Шифр документа

`<номер>-<раздел><подраздел>.<тип>` ИЛИ `<...>-<тип>`. **Разделитель (точка/дефис)
непоследователен между разделами** — смотреть на шифр .СО раздела как образец.
В ВОР главный штамп иногда хранит базовый шифр без разделителя, добавку — с
разделителем в начале (несимметрично с ПНР).
